### Jekyll轻量级极简博客

#### 第一步：点击 [fork](https://github.com/Liberxue/liberxue.github.io#fork-destination-box)
#### 第二步：修改_config.yml 中的 url 为 GitHub 用户名.github.io
#### OK，搭建好了，就这么简单。
#### 我不要任何版权也不需要您捐款，帮忙 star，给个🌟就行谢谢 😄

- [x] 自动生成json搜索
- [x] 自适应模板
- [x] 自动生成feed.xml
- [x] 自动生成分页
- [x] 修改_config.yml 的links 为您的菜单
